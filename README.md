💍 Royal Wedding Invitation Card – Tkinter GUI

A beautifully designed digital wedding invitation card built using Python and Tkinter.
This project showcases GUI design skills with elegant typography, decorative floral borders, and a romantic wedding layout.

📌 Project Overview

This application creates a wedding invitation window featuring:

🌸 Floral decorative borders

💕 Romantic tagline

👑 Highlighted couple names

📅 Wedding date & time

📍 Venue details

💌 Blessing message

🎉 Reception information

The design uses a soft pink theme with red accents to create a premium invitation feel
